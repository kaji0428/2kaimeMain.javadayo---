<ソフトの機能紹介>
BMI計算ソフトは、身体の健康を把握するための便利なツールです。このソフトウェアは、
身長と体重の単純な入力を通じて、BMI（Body Mass Index）を瞬時に計算し、ユーザーに的確な健康情報を提供します。
BMIは、個々の身体の体重と身長の比率を示す指標であり、健康な体重管理や生活習慣の評価に役立ちます。
-----------------------------------------------------------------------------------------------------------
<使い方>
1. 1人目の身長をｍで入力
2. １人目の体重をkgで入力
3. 表示された数字がＢＭＩとなります。
4-6. ２人目(1-3と同様に行う)
